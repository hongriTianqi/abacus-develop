434 //5. cal_gvepsl : gvepsl is used for training with stress label, which is derivative of
435 // descriptors wrt strain tensor, calculated by
436 // d(des)/d\epsilon_{ab} = d(pdm)/d\epsilon_{ab} * d(des)/d(pdm) = gdm_epsl * gvdm
437 // using einsum
438 //6. cal_gvdm : d(des)/d(pdm)
439 // calculated using torch::autograd::grad
440 //7. load_model : loads model for applying V_delta
441 //8. cal_gedm : calculates d(E_delta)/d(pdm)
442 // this is the term V(D) that enters the expression H_V_delta = |alpha>V(D)<alpha|

445 //10. cal_orbital_precalc : orbital_precalc is usted for training with orbital label,
