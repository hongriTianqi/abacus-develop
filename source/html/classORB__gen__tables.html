(1) get distance between R1 and R2 (a.u.) judge if there exist overlap

if distance == 0,

dr independent of R if R == 0. distance += tiny1 avoid overflow during calculation.

(2) if there exist overlap, calculate the mesh number between two atoms

(3) Find three dimension of 'Table_S' or 'Table_T'. -dim1 : type pairs, -dim2 : radial orbital pairs, -dim3 : find lmax between T1 and T2, and get lmax*2+1

Parameters

[in]dtype[in] derivative type, 'S' for overlap, 'T' for kinetic energy, 'D' for descriptor in deepks
