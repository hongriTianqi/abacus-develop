Detailed Description

The tested functions: i. Parallel_Kpoints::init_pools() is the public interface to call the private function Parallel_Kpoints::divide_pools(), which divide all processes into KPAR groups. ii.Parallel_Kpoints::kinf() is the public interface to call another three functions: get_nks_pool(), get_startk_pool(), get_whichpool(), which divide all kpoints into KPAR groups. The default number of processes is set to 4 in parallel_kpoints_test.sh. One may modify it to do more tests, or adapt this unittest to local environment.
