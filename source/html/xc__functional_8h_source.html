103 // This file contains wrapper for the LDA functionals
104 // it includes 3 subroutines:
105 // 1. xc, which is the wrapper of LDA part
106 // (i.e. LDA functional and LDA part of GGA functional)
107 // 2. xc_spin, which is the spin polarized counterpart of xc
108 // 3. xc_spin_libxc, which is the wrapper for LDA functional, spin polarized
109 
110 // NOTE : In our own realization of GGA functional, the LDA part
111 // and gradient correction are calculated separately.
112 // The LDA part is provided in xc, while the gradient correction is
113 // provided in gradcorr through gcxc/gcx_spin+gcc_spin.
114 // While in LIBXC, the entire GGA functional is provided.
115 // As a result, xc/xc_spin and xc_spin_libxc are different for GGA,
116 // the former gives nonzero result, while the latter returns 0.
117 // Furthermore, the reason for not having xc_libxc is that something like
118 // xc_libxc which evaluates functional for individual grid points
119 // is not efficient. For nspin = 1 and 2, v_xc_libxc evaluates potential
120 // on the entire grid. I'm having xc_spin_libxc because v_xc_libxc
121 // does not support nspin = 4.
